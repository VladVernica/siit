Tema JavaScript / Vlad Vernică

Tema 01

O funcție care compara 2 valori si returneaza true daca sunt egale si false daca nu sunt egale.


Tema 02

O funcție care compara 2 valori si returneaza -1 daca primul e mai mic decat al doilea, 0 daca sunt egale si 1 daca primul e mai mare decat al doilea.


Tema 03

O funcție care compara 2 valori si returneaza maximul dintre cele 2.


Tema 04

O funcție care compara 2 valori si returneaza minimul dintre cele 2.


Tema 05

O functie care intoarce suma primelor N numere naturale pozitive.


Tema 06

O funcție care verifica daca N este numar prim (restul impartirii la 1 si la N == 0).


Tema 07

O funcție care intoarce suma primelor N numere prime.


Tema 08

O funcție care primeste un parametru de tip string si intoarce inversul acestuia (abc => cba).


Tema 09

O funcție care intoarce produsul primelor N numere impare pozitive.


Tema 10

O funcție care primeste 2 parametri (arr - array de nr intregi si x - numar) si verifica daca x exista in array.


Tema 11

O funcție care intoarce maximul dintr-un array primit ca parametru.


Tema 12

O funcție care intoarce suma dintre max si min dintr-un array primit ca parametru.


Tema 13

O funcție care verifica daca exista duplicate intr-un array primit ca parametru.


Tema 14

O funcție care intoarce produsul numerelor pozitive dintr-un array primit ca parametru.


Tema 15

O funcție care verifica daca un string primit ca parametru este palindrom (inversul == originalul, ex: abcba == abcba, abca != acba).
